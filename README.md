# TF-IDF Analysis Tool

Инструмент для анализа значимости слов в документах с использованием модели TF-IDF.

## Функционал

- Статистика по слову во всех документах
- Статистика слова в конкретном документе
- Статистика по документу (общее/уникальные слова, топ-5 слов)
- Поиск релевантных документов по ключевым словам

## Запросы

- `WORD <word>` - статистика по слову
- `WORD_IN_DOC <word> <document>` - статистика слова в документе
- `DOC <document>` - статистика по документу
- `QUERY <word1> <word2> ...` - поиск документов
